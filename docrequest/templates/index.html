About the System

System Purpose

The Document Request System is designed to digitize and streamline the process of requesting academic documents. It eliminates the need for physical visits to the registrar's office, reduces processing time, and provides real-time tracking of document requests. The system supports both online payments (simulated GCash/Maya) and cash-on-pickup options for student convenience.

Scope & Limitations
